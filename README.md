# sillycat-wasm-solution
wasm, wasi, rust, python, assemscript, sonarque and etc

### wasi-interface 
It is the WITX interface files and etc

### wasi-impl
It is the implementation of the WITX and runtime to run it

### wasi-consumer-rust
Rust use external to call the impl, build the wasm

### wasi-consumer-as
AssemblyScript use external to call the impl, build the wasm

### wasi-consumer-python
Python use external to call the impl, build the wasm
